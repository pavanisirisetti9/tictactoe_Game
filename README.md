# tictactoe_Game
A simple implementation of the classic Tic Tac Toe game built using modern programming principles. This project recreates the timeless two-player strategy game where players take turns marking spaces on a 3×3 grid, aiming to place three of their marks in a horizontal, vertical, or diagonal row.
